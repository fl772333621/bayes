: 两部烂片。我怀疑什么观众在看完这两个片子之后屁股上能不长坐疮。共同特点
: 是：
: 
: - 导演不懂得“少既是多”的道理，塞进无数的感情冲击，把全片变成一场怪胎
: 秀。情节剧式的段落不停的反复罗嗦，作用雷同的戏剧冲突没完没了的来了一遍
: 又一遍，不但对情节和角色的发展毫无推动作用，反而让影片变得无聊而冗长。
: 这不是合格的导演，反倒是典型的所谓hack。
: 
: - 腻歪的对话，每个人都像作心理治疗一样没完没了的滔滔不绝的谈论自己的感
: 情，虚假造作到三流言情片的水平。这样不但让原本就已经脸谱化的人物变成可
: 笑的说教者，角色个体的变化历程根本不能令人信服。
: 
: - 糟糕到家的表演，每个演员都使尽了吃奶的劲儿，表情狰狞扭曲，冲着对方大
: 喊大叫。这是琼瑶剧里的马景涛——典型的overacting，可以拿来做反面教材用。
: 
: - 缺乏一个哪怕是合格的剪辑师。整体的步伐让人感到冗长不说，在动作场面上
: 更是一片混乱：特别是&lt;太极旗&gt;，在摄影上拼命学大兵瑞恩，可是到了剪辑上却
: 狗屁不通，一到近战场面就只剩下shaky camera，完全看不出战斗是如何进行的。
: 
: - 一方面充满狭隘的民族主义，另一方却又想要抨击过往的政治。对时代背景的
: 肤浅诠释使得整个影片像一场廉价的政治攻击，而对真正重要的东西——造成这
: 些事件的民族/社会原因、产生的影响等等，没有任何挖掘。
